# 100 Days of AI Projects

Welcome to my **100 Days of AI Projects** journey.

I started this repository to track my progress as I learn and build with AI every day for 100 days.  
My goal is to improve step by step by creating small projects, experimenting with new ideas, and documenting everything I learn along the way.

## About This Journey

This is my personal daily challenge where I will:

- Build something related to AI every day
- Learn by doing and improve consistently
- Share my progress openly
- Document each project and what I learn from it

## Projects So Far

### Day 1: YouTube Learning Agent
My first project was the **YouTube Learning Agent**, an AI-powered tool built to help transform YouTube content into useful study material and learning support.

### Day 2: RBAC-RAG Chatbot Project
My next project is the **RBAC-RAG Chatbot Project**, where I will continue exploring AI by building a chatbot with role-based access control and retrieval-augmented generation features.
